## Description

This database compiles key economic indicators and the main influencing factors for Venezuela's economy during 2024–2025, based on data from Perplexity.ai and cited sources. The dataset covers macroeconomic indicators including GDP, inflation, unemployment, oil production, exchange rates, poverty, and public debt.

Key influencing factors analyzed include: oil sector performance, hyperinflation aftermath, migration crisis, U.S. sanctions, informal economy, humanitarian situation, and dollarization.
